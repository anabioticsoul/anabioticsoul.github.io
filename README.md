# anabioticsoul.github.io

A personal interactive portfolio website built with **React**, **Vite**, **Three.js**, and **React Three Fiber**.

This project presents my work in an explorable 3D space environment. Instead of using a traditional flat portfolio layout, the site is designed as an interactive cosmic scene where visitors can navigate through different sections such as **About**, **Projects**, **Publications**, and **Contact**.

## Features

- Interactive 3D portfolio experience
- Space-themed scene with planets, skybox, and imported 3D assets
- Built with React and Vite for fast development and deployment
- Powered by Three.js through React Three Fiber
- Modular structure for extending sections and visual elements
- Deployed through GitHub Pages

## Tech Stack

- **React**
- **Vite**
- **Three.js**
- **@react-three/fiber**
- **@react-three/drei**

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/anabioticsoul/anabioticsoul.github.io.git
cd anabioticsoul.github.io
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

### 5. Preview the production build

```bash
npm run preview
```

## Deployment

This repository is intended for deployment with **GitHub Pages**.

Because the repository name is `anabioticsoul.github.io`, it is treated as a **user site**, so the website is served from the root domain:

```text
https://anabioticsoul.github.io/
```

## Content Overview

This website is designed as a personal portfolio in a navigable 3D universe.  
Its purpose is to combine research, projects, publications, and personal presentation in a more immersive format.

Typical sections include:

- **About** — personal introduction and background
- **Projects** — selected works and ongoing development
- **Publications** — papers, datasets, and research output
- **Contact** — ways to connect
